# como vincular um repositório do GITHUB com o Visual Code
1 no GITHUB abra o repositório e clique no botao CODE -> local -> copiar link do 
repositório
2 no Visula Code abra a pasta de seus projetos ( exemplo: Documentos/Python/ )
3 Abra o terminal e gite o comando: git clone <cole aqui o link copiado>
4 tecle Enter
5 No VS code abra a pasta clonada
6 No terminal digite: 
--global git config user.name <digite aqui seu nome de usuario no Github>
7 tecle Enter
8 ainda no terminal digite:
 --global git config user.email <digite aqui seu email no Github>
9 tecle Enter
